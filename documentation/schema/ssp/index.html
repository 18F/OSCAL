        OSCAL Implementation Layer: System Security Plan (SSP) Model

        The OSCAL System Security Plan (SSP) model represents a description of the control implementation of an information system. The SSP model is part of the OSCAL Implementation layer.

        This section contains the following topics:

        • System Security Plan JSON Model Map: An outline of the OSCAL System Security Plan model JSON format.
        • System Security Plan JSON Schema Reference: Object-level definitions of the OSCAL System Security Plan model JSON format.
        • System Security Plan XML Model Map: An outline of the OSCAL System Security Plan model JSON format.
        • System Security Plan XML Schema Reference: Element-level definitions of the OSCAL System Security Plan model XML format.
