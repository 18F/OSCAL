OSCAL Catalog Layer: Catalog Model

The OSCAL Catalog model represents the information contained within a control catalog. The Catalog model is the sole model in the OSCAL Catalog layer.

The NIST SP 800-53 rev 4 control catalog is provided as an example in the OSCAL GitHub repository in XML, JSON, and YAML formats.

This section contains the following topics:

  • Catalog JSON Model Map: An outline of the OSCAL Catalog model JSON format.
  • Catalog JSON Schema Reference: Object-level definitions of the OSCAL Catalog model JSON format.
  • Catalog XML Model Map: An outline of the OSCAL Catalog model JSON format.
  • Catalog XML Schema Reference: Element-level definitions of the OSCAL Catalog model XML format.
